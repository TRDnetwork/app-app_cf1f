# Deploy Instructions: Dawn Brew

## Deploy to Vercel
1. Push this code to a new GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in with your GitHub account
3. Click "New Project" and import the repository
4. Vercel will automatically detect this as a static site — no configuration needed
5. Click "Deploy" — your site will be live at a `.vercel.app` URL within minutes

## Environment Variables
None required. This is a fully static site with no backend dependencies.

## First-time Setup
None required. All assets and functionality are self-contained:
- Client-side email form validation only (no actual submission)
- Responsive design works across all devices
- Social links open in new tabs
- Dark theme with amber accents automatically applied

> ✅ Tip: After deployment, set your custom domain in Vercel settings if desired.
