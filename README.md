# Dawn Brew — Coming Soon Landing Page

A minimalist, dark-themed landing page for **Dawn Brew**, a hand-roasted coffee shop opening in Karnal, Summer 2026.

## Features

- **Hero Section**: Tagline and email signup form with client-side validation
- **Content Sections**: Origin, brewing philosophy, and location storytelling
- **Responsive Design**: Mobile-first, works from 320px to 1920px+
- **Dark Theme**: Deep navy background (`#0A1628`) with warm amber accents (`#D97706`)
- **Accessibility**: Semantic HTML, proper contrast, ARIA labels
- **Performance**: No external dependencies, inlined critical CSS, optimized assets
- **PWA Ready**: Includes `manifest.json` and SVG favicon for installability
- **Social Links**: Instagram and Twitter links open in new tabs with `rel="noopener"`

## Design

- **Aesthetic**: Dark, moody minimalism with warm amber accents
- **Typography**: 
  - Headings: *Playfair Display* (serif, elegant)
  - Body: *Inter* (sans-serif, readable)
- **Animations**: Scroll-triggered fade-in effects using Intersection Observer
- **Interaction**: Hover states on buttons and links with smooth transitions

## Usage

1. Open `index.html` in any browser
2. Enter an email in the form to see validation and success message
3. Scroll to view animated section entries

## No Build Step

This is a static site with no framework, no bundler, no dependencies. Just open and go.
